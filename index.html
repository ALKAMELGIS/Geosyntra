Agro Cloud

تظهر هذه الصفحة عندما يكون مصدر GitHub Pages هو «Deploy from a branch» من جذر المستودع.

التطبيق (React/Vite) يُبنى في frontend/dist ويُنشر عبر GitHub Actions فقط.

  1. افتح المستودع على GitHub → Settings → Pages.
  2. ضمن Build and deployment اختر المصدر: GitHub Actions (وليس Deploy from a branch).
  3. ادفع أي تعديل إلى main أو شغّل يدوياً سير العمل Deploy to GitHub Pages.

بعد ضبط المصدر على Actions، افتح الواجهة من الرابط (مع # لأن التطبيق يستخدم HashRouter):

https://alkamelgis.github.io/AgroCloud/#/

الوثائق الكاملة للمستودع: REPOSITORY.md (ليس عرضاً على الموقع عند استخدام Actions).

English: You are seeing this file because Pages is publishing the repository root instead of the Vite build. Switch Settings → Pages → Source to GitHub Actions, then push to main or run the Deploy to GitHub Pages workflow. The published app lives in frontend/dist.

After switching to Actions, open the app (hash route): https://alkamelgis.github.io/AgroCloud/#/
