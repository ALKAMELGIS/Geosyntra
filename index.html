AgroCloud

هذه الصفحة تظهر فقط عندما يكون GitHub Pages مضبوطاً على فرع main بدلاً من فرع البناء. التطبيق الفعلي يُنشر على فرع gh-pages بواسطة سير العمل Deploy to GitHub Pages.

ما الذي تفعله:

  1. افتح Settings → Pages للمستودع.
  2. تحت Build and deployment: المصدر Deploy from a branch.
  3. اختر الفرع gh-pages والمجلد /(root) ثم احفظ.
  4. بعد دقيقة، جرّب https://alkamelgis.github.io/AgroCloud/#/ (HashRouter).

English: This notice appears when Pages is set to publish from main. The SPA is pushed to the gh-pages branch by CI. In Settings → Pages, choose branch gh-pages, folder /(root), then open the live app.
