  1. Settings → Pages → Build and deployment — المصدر Deploy from a branch — الفرع gh-pages — المجلد / (root).

  3. لا تختر فرع main للنشر العلني — سيُعرض هذا الملف التوضيحي بدلاً من التطبيق.

English: This stub is served when GitHub Pages publishes the main branch at / (root). The workflow builds frontend/dist and pushes the site to the gh-pages branch. Optional auto-fix: add repo secret PAGES_ADMIN_TOKEN (see README) so the deploy workflow can call the GitHub API to point legacy Pages at gh-pages.
